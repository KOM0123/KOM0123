### 소개

로봇을 움직이는 코드를 짜는 걸 좋아하는 로보틱스 엔지니어입니다.
TurtleBot3, Limo, 매니퓰레이터로 자율주행·로봇팔 제어를 실험하고, ROS2 위에서 시뮬레이션부터 실제 하드웨어까지 연결하는 걸 즐깁니다.
